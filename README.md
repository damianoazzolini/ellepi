# ELLEPI

## Installation
```
python3 -m pip install .
```

## Example usage
time ellepi -f $filename -v 1 -ec $ec -p $p -rpi $rpi -age 0.3 --train 1 2 3 4 --test 5 -par 0 -pdr 0
